# Environment contract

The validated baseline is Xilinx Vivado and SDK 2018.3 on Windows.

Default installation roots:

- `E:\Xilinx\Vivado\2018.3`
- `E:\Xilinx\SDK\2018.3`

The connected adapter currently enumerates as FTDI VID `0403`, PID `6011`.
This identifies a multi-channel FTDI interface, not the carrier board or FPGA
pinout.
